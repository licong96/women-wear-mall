# 香菇街 #
----------
## 介绍 ##

> 2017年10月国庆期间用业余时间开发的一个纯前端项目。2018年8月调整了一下接口和一些功能。
我用jsonp抓取了蘑菇街的数据来渲染页面，可惜，有些数据有签名限制，抓取不到，导致没法继续做下去了。
过了这么久，现在再来看这个项目，发现了很多当年没有发现的问题，一些比较糟糕的地方。但是我不准备修复这些问题，我会把它当做成长的一部分，紧记当年的自己是多么的愚蠢。
